Position Sizing

Posted on 2026-04-22

选股 vs. 系统

选股：

  • 极强专业能力
  • 大量时间精力
  • 高风险、精神折磨

系统：

  • 只需纪律，不需天赋
  • 每周 1-2 小时足够
  • 风险可控、心态稳定

「指数 + 期权」双轨交易系统

上轨：QQQ 指数定投，长期复利增长
下轨：QQQ Wheel + LEAPS，现金流，利润加速

$2W -> $10W -> $100W -> 退休

第一阶段：起步期

$2W -> $10W
核心武器：QQQ Sell Put (CSP)
预计时间：3-5 年（含工资持续投入）

为什么用 QQQ 做 Sell Put

直接持有 TQQQ：震荡损耗严重（Volatility Decay）
买 TQQQ LEAPS：杠杆 x 3 x LEAPS 杠杆 = 波动失控
Sell Put QQQ：赚高波动率权利金，不被动持股

Sell Put 核心指标

QQQ 当前价：$49.17
盈亏平衡点：$45 - $2.1 = $42.9，即使 QQQ 暴跌 13%，依然不亏
抵押回报率：$210 / $4,290 = 4.9%（34 天），月度范围：3%-6%（随 IV 浮动）

起步期仓位策略

$20,000 -> $4,290 一张 CSP 保证金 + $15,710 现金后备军
目标：完成 10 轮完整循环，到期收租或被行权 -> 卖 Call 闭环，都算成功
起步期可超配，账户变大后回归正常比例

Wheel Strategy

  1. Sell Put，收 $210 权利金
  2. 被行权，持股成本 $42.9
  3. Covered Call，收 $180 权利金
  4. Called Away，获利出场

车轮永不停转，每一圈都在收钱
被行权期间积极卖 Call = 对冲震荡损耗

高胜率开仓三步曲

  1. 看底气：大趋势向上
    确认 QQQ 价格稳稳站在 200 日均线之上，代表长牛格局
  2. 等洗盘：RSI 跌破 50
    买盘枯竭，散户恐慌抛售，出现健康的洗盘回调
  3. 迎暴击：明显下跌阴线
    恐慌导致 IV 飙升，此时 Sell Put 权利金最丰厚

被行权后的两种局面

QQQ > 200 SMA
黄金坑
大趋势护盘
放心卖 Covered Call
等待反弹获利出场

QQQ < 200 SMA
接飞刀
立刻止损保命
或卖 ITM Call 极端防御
绝不用杠杆 ETF 死扛

牛熊分界线 = 你的操作分水岭

第二阶段：加速期

劳动收入 + 投资收入并驾齐驱，3-5 年滚到 $10W
策略必须升级，构建三足鼎立系统
QQQ 底仓 + QQQ Wheel 现金流 + QQQ LEAPS 爆发力

QQQ LEAPS

Delta 0.7，到期日 2 年，Deep ITM

入场条件：RSI < 35
出场条件：不奢望吃满波段，严格执行铁血纪律，落袋为安：

  • 0-12 月：盈利 100%
  • 12-15 月：盈利 50%
  • 15-18 月：盈利 30%
  • 18-24 月：绝对禁区，无条件平仓

LEAPS 风险控制

风险场景：QQQ -30% -> LEAPS -60+%
仓位控制：期权池 20% = 总资产 8%，即使归零 -> 仅损失 8%

敢拿、拿得住、亏得起

利润流向

Wheel + LEAPS 期权利润 -> QQQ 底仓
底层资本越大，你的系统越稳

完美资产配比公式

年龄 + 20 = 正股底仓占比
现金仓位 10%，VIX > 30 时主动抄底

总仓位配比

50% QQQ 底仓，稳定
10% 现金，防守
32% QQQ Wheel，现金流
8% QQQ LEAPS，爆发力

底仓保稳定，Wheel 保现金流，LEAPS 保爆发力

压力测试：2022 极端熊市

满仓 TQQQ：-79%，账户几乎归零
本系统仓位配比：

  • 50% QQQ -> -16.5%
  • 32% Wheel -> -16%（CC 回血）
  • 8% LEAPS -> -6.4%
  • 10% 现金 -> 抄底弹药

总回撤 ~= -25%-35%
2023 纳指反弹 +55% -> 全部收回 + 盈余

亏了能活，活了能回来

没有 $2W 怎么办

  1. 审视赚钱能力，专研一门高收入技艺
  2. 开源节流，降低消费欲望
  3. 省吃俭用攒下第一桶金

第一桶金靠的不是智商，而是自律和汗水

第三阶段：终极思维

$100W 之后的终极配置：

  • 50%-60%：QQQ + 现金：家族传承底仓，抵抗通胀
  • 8%：LEAPS：牛市周期的 Alpha 增益器
  • 32%: Wheel：人造退休金，源源不断的现金流

资金在涨 + 现金流不断 = 安心退休

年龄增长，仓位自然过渡

30 岁：正股 50%，期权池 40%
40 岁：正股 60%，期权池 30%
50 岁：正股 70%，期权池 20%

$100W x 32% = $32W vs. $200W x 25% = $50W
占比虽小，绝对金额增长 -> 期权现金流不减反增

可复制的百万美金系统

  1. 攒 $2W 门票
  2. QQQ Sell Put + 每月存钱
  3. $10W 后引入 LEAPS
  4. 年龄 + 20 构建底仓
  5. 百万后仓位养老

慢就是快，用几年的纪律，换后半生的自由
